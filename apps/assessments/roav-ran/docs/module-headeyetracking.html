Module: headeyetracking

This module provides utilities for facial landmark detection, eye tracking, and geometric calculations. It includes functions for processing FaceMesh results, drawing face contours, calculating head metrics, and preprocessing image data for machine learning models. The module also contains constants for facial landmark indices and default configurations for device and participant settings. Key features: - FaceMesh landmark processing - Eye tracking initialization and visualization - Image preprocessing for ML models - Geometric calculations (e.g., polygon area, centroids) - Utilities for comparing face shapes and positions
