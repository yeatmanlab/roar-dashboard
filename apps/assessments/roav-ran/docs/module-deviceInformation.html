(static) getDeviceInfo() → {Object}

Retrieves device and screen information from the browser.
Properties:
Name Type Description
device Object Basic device information retrieved from the `navigator` object.
Properties
Name Type Description
userAgent string The user agent string of the browser.
platform string The platform on which the browser is running.
language string The preferred language of the user.
deviceMemory string | number The amount of device memory (in GB), or "Not available" if unsupported.
hardwareConcurrency number The number of logical processor cores.
maxTouchPoints number The maximum number of touch points supported.
cookiesEnabled boolean Indicates whether cookies are enabled.
webDriver boolean Indicates whether the browser is controlled by WebDriver automation.
onlineStatus boolean Indicates whether the browser is currently online.
gpu string The detected GPU renderer or "Not available" if unavailable.
screen Object Screen and display-related information.
Properties
Name Type Description
width number The screen width in pixels.
height number The screen height in pixels.
colorDepth number The number of bits used to represent colors.
pixelDepth number The number of bits per pixel.
retinaDisplay boolean Indicates if the screen is a Retina display (device pixel ratio ≥ 2).
landscapeOrientation boolean Indicates if the screen is in landscape orientation.
